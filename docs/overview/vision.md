# VibeForge Vision

VibeForge adalah AI Coding Workspace yang menyatukan proses planning, task management, coding, review, logs, dan project context.

Tujuannya adalah mengurangi context switching antara:

- VS Code
- Kanban board
- AI chat
- Google Docs/Notion
- Terminal
- GitHub
- Logs/reporting tools

VibeForge harus membuat user bisa:

1. Membuat project.
2. Meminta AI memecah pekerjaan besar.
3. Mengubah planning menjadi schedule beberapa hari.
4. Convert schedule menjadi task Kanban.
5. Membuka task pending di Workspace.
6. Melihat file project.
7. Mengedit code.
8. Menjalankan AI agent.
9. Melihat terminal, diff, logs, dan hasil testing.
10. Menyimpan daily/weekly logs otomatis ke NocoDB.

VibeForge adalah gabungan:

```txt
VS Code + Linear + GitHub + Notion + AI Agent
```
