# ADR 0002: README Is For Humans

## Status

Accepted

## Context

VibeForge will be open source, so README must explain installation and usage, not act as AI constitution.

## Decision

README.md is for human users/contributors.
AI rules live in AGENTS.md, CLAUDE.md, .clinerules, and docs/ai.

## Consequences

- Open-source users get a clean README.
- AI agents still get strict instructions.
