# Definition of Done

A task is Done only if:

- Acceptance criteria are complete.
- No blocker remains.
- Build passes.
- Typecheck passes.
- Lint passes.
- UI loading state exists.
- UI empty state exists.
- UI error state exists.
- Toast/feedback exists.
- Destructive actions have confirmation.
- NocoDB data is saved if relevant.
- Daily log is created.
- Project context is updated if needed.
- No old branding remains.
- No secret is leaked.
- No critical TODO remains.
