# Planner and Schedule UX

## Planner Purpose

Planner converts a large objective into structured development plan.

Input:

- Objective
- Scope
- Complexity
- Available days
- Preferred task size
- Deadline optional
- Related docs

Output:

- Plan
- Risks
- Dependencies
- Suggested tasks
- Day breakdown
- Acceptance criteria

## Schedule Purpose

Schedule breaks work across days.

It does not always require exact dates.

Use:

- Day 1
- Day 2
- Day 3

If user provides dates, use dates.

## Convert to Tasks

Planner must include action:

- Convert selected plan items to Kanban tasks
- Save to NocoDB
