# Kanban UX

## Columns

- Backlog
- Todo
- In Progress
- Review
- Testing
- Done
- Blocked

## Card Content

Each card should show:

- Title
- Priority
- Type
- Estimate
- Assigned agent
- Provider/model
- Progress
- Dependencies
- Related files
- Branch
- Blocker indicator

## Task Detail

Task detail drawer should include:

- Description
- Acceptance criteria
- Checklist
- Related docs
- Related files
- Plan
- Schedule
- Logs
- Agent runs
- Blockers

## Rule

Do not mark task Done if blocker exists.
