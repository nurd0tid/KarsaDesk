# Agent Operating Loop

Agent should repeat this loop:

1. Read context.
2. Identify active task.
3. Validate acceptance criteria.
4. Use Context7 for packages.
5. Use Sequential Thinking for complex decisions.
6. Plan.
7. Save plan.
8. Implement.
9. Review.
10. Test.
11. Log.
12. Update context.
13. Update task.
14. Pick next task.

This loop prevents report-only AI behavior.
